# 团本划水检测 Raid Water Check

## 版本

0.1.0

## 插件简介

团本划水检测是一个魔兽世界团本表现复盘插件。它会在 Boss 战结束后生成团队表现摘要，帮助团长查看死亡、长时间无动作、低贡献、承伤异常、可规避伤害、打断、驱散和关键技能使用情况。

插件默认只记录 Boss 战；如需记录小怪或非 Boss 战，可在设置中开启“也记录小怪/非Boss战”。

## 主要功能

- Boss 战后自动生成表现报告
- 可视化设置面板，分为基础设置和高级设置
- 右上角插件集合入口和小地图备用按钮
- 统计伤害、治疗、承伤、死亡、打断、驱散、施法次数
- 统计长时间无动作
- 可自定义可规避伤害技能
- 可自定义关键技能检查
- 点击玩家查看个人详情
- 手动通报异常玩家到团队/队伍频道

## 常用命令

```text
/rwc
/huashui
/rwc show
/rwc options
/rwc announce
/rwc trash
/rwc rules
/rwc avoid add 技能ID 技能名
/rwc track add 技能ID 技能名 [职业英文] [最低次数]
```

## 适配说明

当前 `.toc` 使用正式服 Interface 号。怀旧服如果提示过期，可勾选“加载过期插件”，或按对应客户端版本修改 `RaidWaterCheck.toc` 里的 `Interface` 数字。

## 合规说明

- 不含自动战斗、自动施法、自动喊话循环等自动化功能。
- 不修改游戏客户端文件。
- 不联网，不上传战斗数据。
- 只读取游戏插件 API 提供的战斗日志事件。
- 团队通报必须由玩家手动点击按钮或输入命令触发。
