# 团本划水检测 Raid Water Check

一个轻量级魔兽世界团本表现复盘插件。插件会在 Boss 战结束后生成团队表现摘要，帮助团长快速查看异常表现、可规避伤害、死亡、打断、驱散、承伤和关键技能使用情况。

插件定位是“复盘辅助”，不是自动定罪工具。DPS 低可能来自机制分工、转火、跑位、死亡、装等、职业环境或团队策略，需要结合实际战斗判断。

## 安装

把整个 `RaidWaterCheck` 文件夹复制到魔兽目录：

```text
World of Warcraft/_retail_/Interface/AddOns/RaidWaterCheck
```

如果是怀旧服，把它复制到对应版本的 `Interface/AddOns` 目录，并按当前版本改 `RaidWaterCheck.toc` 里的 `Interface` 数字。

## 使用

进入游戏后输入：

```text
/rwc
/huashui
```

常用命令：

```text
/rwc show   显示上一场报告
/rwc menu   打开操作面板
/rwc summary 显示当前/上一轮副本总结
/rwc endrun  手动结束当前副本会话并显示总结
/rwc options 打开可视化设置
/rwc announce 通报低分异常到队伍/团队频道
/rwc rules  查看当前可规避伤害规则
/rwc avoid add 技能ID 技能名  添加可规避伤害
/rwc avoid del 技能ID  删除可规避伤害
/rwc track add 技能ID 技能名 [职业英文] [最低次数]  添加关键技能检查
/rwc track del 技能ID  删除关键技能检查
/rwc minimap  显示/隐藏小地图按钮
/rwc auto   切换战斗结束后自动弹报告
/rwc trash  切换是否记录小怪/非Boss战
/rwc reset  重置统计
/rwc demo   生成演示记录，用来测试报告和总结排版
/rwc start  手动开始测试
/rwc stop   手动结束测试
```

## 可视化设置

输入 `/rwc options` 可以打开设置窗口，设置分为“基础设置”“高级设置”“权重设置”三个页签，目前可调整：

- 是否战斗结束后自动弹报告
- 是否记录小怪/非 Boss 战
- 是否启用施法频率扣分
- 是否启用贡献占比扣分
- 是否启用承伤占比扣分
- 是否启用可规避伤害扣分
- 是否启用关键技能缺失扣分
- 最短记录战斗时长
- 无动作宽限时间
- 报告显示人数
- 通报分数线
- 轻度/中度/重度无动作阈值
- 高承伤占比阈值
- 可规避伤害扣分
- 缺失关键技能扣分
- 是否启用 NAXX/WCL 冲分参考提示

## 当前会统计

默认只记录 Boss 战。打开“也记录小怪/非Boss战”后，插件才会记录普通进战斗到脱战的战斗。

插件会把同一个副本会话内的多个 Boss 战整合成副本总结。第一个 Boss 开始时自动开启副本会话；离开副本时自动生成总结，也可以输入 `/rwc endrun` 手动结束，或输入 `/rwc summary` 随时查看当前总结。

- 伤害量
- 有效治疗量
- 施法成功次数
- 打断次数
- 驱散/偷取次数
- 死亡次数
- 长时间无动作估算
- 承伤来源
- 可规避伤害次数和金额
- 自定义关键技能缺失检查
- 个人详情页，点击报告里的玩家即可打开
- 副本总结页，整合多个 Boss 的平均分、最低分、总死亡、总可规避伤害、打断驱散和主要异常
- 综合异常分数
- NAXX/WCL 冲分参考提示：标记可能不计入 WCL 排名的目标伤害或特殊 Buff

## 可规避伤害规则

插件不会凭空知道每个 Boss 的“地板技能”应该怎么判定。你可以边打边补规则：

```text
/rwc avoid add 123456 地板火
/rwc avoid del 123456
/rwc rules
```

添加后，之后战斗里吃到这个技能的玩家会记录为“可规避伤害”，并按设置窗口里的扣分规则扣分。

## NAXX/WCL 参考

基础设置里可以启用“NAXX/WCL 冲分参考提示”。开启后，插件会在 NAXX Boss 战里额外标记部分 WCL 规则可能排除的小怪伤害，以及 Battle Squawk、Fungal Bloom、Silithyst、塔迪乌斯正/负极电荷等可能影响冲分判断的 Buff。

这个模式不会替代 WCL 官方结算，也不会改变团长复盘分数；它只是在游戏内提醒“这部分数据上传 WCL 后可能不是有效冲分数据”。最终排名仍以 Warcraft Logs 上传后的官方解析为准。

## 右上角入口

新版客户端的右上角插件集合里会出现团本划水检测图标：

- 左键：打开上一场报告
- 右键：打开设置

小地图旁边也会显示一个备用图标：

- 左键：打开上一场报告
- 右键：打开设置
- 拖动：移动图标位置
- `/rwc minimap`：显示或隐藏按钮

关键技能规则同理，适合标记爆发、团队减伤、战复、嗜血等你们团希望复盘的技能：

```text
/rwc track add 31884 复仇之怒 PALADIN 1
/rwc track add 80353 时间扭曲 MAGE 1
/rwc track del 31884
```

职业英文使用游戏里的英文职业 token，例如 `PALADIN`、`MAGE`、`DRUID`。不填职业时会对所有人检查，建议谨慎使用。

## 重要说明

- 插件不会自动操作角色。
- 插件不会修改游戏客户端文件。
- 插件不会上传任何个人数据。
- 插件只在本地读取游戏提供的战斗日志事件并生成界面统计。
- 通报功能只会在玩家手动点击“通报异常”时发送团队/队伍消息。

Retail 新版本如果限制战斗日志 API，游戏内统计会不完整。那时更适合做“插件内轻量提示 + 外部 WoWCombatLog 分析器”的双层方案。
