# 新手盒子发布准备

## 建议标题

团本划水检测 Raid Water Check

## 一句话简介

Boss 战后自动生成团队表现复盘报告，帮助团长查看死亡、无动作、可规避伤害、打断驱散、承伤异常和关键技能使用情况。

## 详细介绍

团本划水检测是一个魔兽世界团本表现复盘插件。插件默认只记录 Boss 战，在战斗结束后生成团队表现摘要，帮助团长快速定位需要复盘的异常表现。

插件会统计伤害、有效治疗、施法次数、打断、驱散、死亡、承伤、长时间无动作、可规避伤害和自定义关键技能使用情况。报告中可以点击玩家查看个人详情。

插件支持可视化设置面板，包含基础设置和高级设置。玩家可以选择是否记录小怪/非 Boss 战，调整通报分数线、显示人数、无动作阈值等参数。

## 审核说明

本插件为本地战斗日志复盘工具：

- 不自动操作角色
- 不自动施法
- 不修改游戏客户端文件
- 不联网、不上传用户数据
- 只读取魔兽世界插件 API 提供的战斗日志事件
- 团队通报需要玩家手动点击按钮或输入命令触发
